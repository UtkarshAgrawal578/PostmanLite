# 🚀 PostmanLite

A lightweight Postman-inspired API testing tool built with **HTML, CSS, JavaScript, Node.js, and Express**. It provides an intuitive interface for testing REST APIs, managing collections, environment variables, and viewing formatted API responses.

## Features

* Send **GET, POST, PUT, PATCH,QUERY and DELETE** requests
* Query Parameters & Custom Headers
* Request Body (JSON, Raw, Form Data)
* Authentication (Bearer Token, Basic Auth, API Key)
* Request Collections
* Request History
* Environment Variables
* Response Status, Headers, Size & Response Time
* JSON Syntax Highlighting
* Backend Relay to bypass CORS restrictions

## Tech Stack

* HTML
* CSS
* JavaScript (ES Modules)
* Node.js
* Express.js

## Run the Project

* **Backend Entry Point:** `server/src/index.js`
* **Frontend Entry Point:** `client/index.html`

Start the backend:

```bash
cd server
node src/index.js
```

Then open `client/index.html` in your browser (or use **VS Code Live Server** for a better development experience).
